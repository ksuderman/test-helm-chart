# Test Helm Chart

A demo Helm chart for testing the release automation process.

![Version](https://img.shields.io/github/v/release/ksuderman/test-helm-chart?label=chart%20version)

## Overview

This repository demonstrates:
- PR validation workflow
- Automated version bumping
- Release creation with auto-generated notes
- Downstream notifications to test-ansible-playbook
- Chart publishing to test-helm-repo
